## 🧠 Sobre o projeto

Este projeto foi desenvolvido durante a realização de um **bootcamp em parceria com a Shopee**, oferecido pela plataforma :contentReference[oaicite:0]{index=0}.

A aplicação foi construída acompanhando as aulas e seguindo as metodologias propostas ao longo do bootcamp, que tem como foco o aprendizado prático por meio de **desafios, projetos reais e aplicação direta dos conceitos estudados**. :contentReference[oaicite:1]{index=1}

O projeto utiliza tecnologias fundamentais do desenvolvimento web, como:
- **HTML**
- **CSS**
- **JavaScript**

Durante o desenvolvimento, foram aplicados conceitos importantes como:
- estruturação de páginas web  
- estilização com CSS  
- lógica de programação com JavaScript  
- manipulação do DOM  

## 🎯 Objetivo

O principal objetivo deste projeto foi **colocar em prática os conhecimentos adquiridos no bootcamp**, reforçando o aprendizado por meio da construção de um projeto real, conforme a proposta da plataforma.

Além disso, o projeto também contribui para a construção de portfólio, já que os bootcamps da DIO incentivam o desenvolvimento de soluções práticas e aplicáveis ao mercado. :contentReference[oaicite:2]{index=2}

## 💡 Considerações

Este projeto representa não apenas um exercício de programação, mas também a aplicação das **metodologias de ensino da DIO**, que priorizam o aprendizado prático, desafios e desenvolvimento de projetos como forma de preparação para o mercado de tecnologia.

Assim, ele demonstra minha evolução durante o bootcamp e minha capacidade de aplicar na prática os conteúdos aprendidos.
